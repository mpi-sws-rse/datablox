This is a the documentation of the message exchanges used in Datablox.
The formatted documementation is in messages.html. To generate the docs
from the sources, use the Makefile. You will need to install docutils
and plotutils.

There is some 3rd party software included for the build:
1. sequence.pic is from UMLGraph (http://www.spinellis.gr/umlgraph/)
2. docutils_basic.css is a Restructured Text theme from
   https://github.com/matthiaseisen/docutils-css.
